# 📖 프로젝트 소개
> 심리 상담 플랫폼인 '어바웃 미'는 익명 커뮤니티/비대면 상담을 제공하여 심리 상담에 대한 접근성을 높이는 것을 목적으로 합니다.<br/>
> 전문가가 등록한 상담 바우처를 클라이언트가 쉽게 비교하여 구매/예약할 수 있으며 상담의 전과정을 비대면으로 진행할 수 있습니다.
>
